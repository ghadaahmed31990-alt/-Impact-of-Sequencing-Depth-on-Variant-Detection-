System Setup (Run Once)This file documents manual system setup steps required before running any scriptsin this project. These steps are not automated by design.---1) Install Docker Desktop (Windows)Go to: https://www.docker.com/products/docker-desktopDownload Docker Desktop for Windows (AMD64)Run the installerDuring installation:✅ Enable "Use WSL2 instead of Hyper-V"Allow Docker to install WSL2 integration automaticallyRestart your system if requested---2) Enable Docker WSL IntegrationOpen Docker DesktopGo to: `Settings → Resources → WSL Integration`Enable integration for your Ubuntu distroClick Apply & Restart---3) Verify InstallationOpen Ubuntu (WSL) and run:```bashdocker --versiondocker run hello-world
